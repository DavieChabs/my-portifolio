## 🎨 Customization Examples

### Change Colors
Edit `tailwind.config.js` line 6-18:
```js
colors: {
  gold: {
    // Update these hex codes
    600: '#YOUR_COLOR',
    400: '#YOUR_COLOR',
  }
}
```

### Add Project
Edit `src/components/Projects.jsx` line 9-45:
```js
{
  title: 'Your Project',
  description: 'What it does',
  technologies: ['React', 'Node.js'],
  // ...
}
```

### Update Skills
Edit `src/components/Skills.jsx` line 10-15:
```js
{
  title: 'Your Category',
  skills: ['Skill1', 'Skill2', 'Skill3'],
}
```
